# Readme
**Code para envio de relatório para setor de empresas**

**Tecnologias Usadas**
- Python


**Passo a passo para construção do código:**

- Passo 1. - Importar base de dados (utilizei planilha excel)
- Passo 2. - Calcular o faturamento de cada loja
- Passo 3. - Calcular a quantidade de produtos vendidos de cada loja
- Passo 4. - Calcular o ticket médio dos produtos de cada loja
- Passo 5. - Enviar Email para a diretoria
- Passo 6. - Enviar Email para cada loja
